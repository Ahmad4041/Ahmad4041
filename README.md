## About Me

I build **production software across the full stack** — from multi-tenant AI agent platforms and solar operations CRMs to Flutter fintech apps and dual-platform Shopify storefronts.

Over the past year I've shipped:

- **Multi-tenant agent PM platforms** (NestJS, Next.js, FastAPI, pgvector) with project memory and governance policies
- **Solar operations CRMs** with drag-and-drop pipelines, team scheduling, and automated alerts
- **E-commerce scraping & publishing pipelines** (Puppeteer, Bull/Redis, PostgreSQL) with AI content enhancement
- **Mobile banking** (Flutter) backed by Java/PHP API monorepos
- **Shopify + WordPress** theme builds with Playwright QA and Lighthouse audits

I care about **reliability**: queues, retries, health checks, type safety, and docs that agents (and humans) can follow.

Currently building solutions at **Software Systems US**. **Master's in Applied AI.** Based in **Melbourne, Australia**.
